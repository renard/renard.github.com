Deprecation

Deprecated forms since o-blog v1

Org backend

Since org-mode backend was the only supported in o-blog v1, some deprecation has been made from version 1 to version 2.

Widgets

Jumbotron

begin_o_blog_hero_unit 
#+begin_o_blog_hero_unit
foo
#+end_o_blog_hero_unit
Use jumbotron instead 
#+begin_jumbotron
foo
#+end_jumbotron

Grid

begin_o_blog_row 
#+begin_o_blog_row 5
Col 1
#+o_blog_row_column  6
Col 2
#+end_o_blog_row
Use #+begin_row and #+begin_col instead 
#+begin_row
#+begin_col: md="6"
Col 1
#+end_col
#+begin_col: md="5"
Col 2
#+end_col
#+end_row
