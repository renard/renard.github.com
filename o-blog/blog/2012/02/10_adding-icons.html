Adding icons

  •  Using modal source code
  •  

Icons from glyphicons are supported support by simply naming the icon using italic style:

Source example 

/icon-calendar/ calendar

Rendered output

icon-calendar calendar

This could also be used for the top navbar using something like:

Source example 

- /icon-book icon-white/ Documentation
