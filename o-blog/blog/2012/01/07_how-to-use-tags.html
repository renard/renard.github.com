How to use tags

  • Creating a blog
  • Example of some org syntax

Tags are useful to classify articles. They are stored in the tags property for each entries. They are not defined as org tags to allows any character but comma (",") to be used.

To add or modify a tag for a post, just use C-c C-x p tags and enter the tag value.

If an article has more than one tag, separate them by a comma (",").
