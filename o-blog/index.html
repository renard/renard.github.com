o-blog publication tool for emacs

Requirements

  • Emacs
  • pandoc or org-mode
  • el-get or git

Setup

If you are using el-get it would be very easy to install. All you need to do is to add o-blog to el-get-sources and type:

M-x el-get-install o-blog

Publish

Only one single command:

M-x o-blog-publish

Please note that this site is for o-blog 2. Old o-blog versions is still available under the o-blog-v1 branch. o-blog 2 has been designed to be fully compatible with o-blog 1 but if you have compatibility problems please open an issue on Github.
